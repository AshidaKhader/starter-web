A very simple online mobile application for Yard Store keepers to record their daily Income and Expenses. 

There are 3 types of user roles
1. Store keeper : Store Keeper can enter the details of daily transactions through the mobile app.

2. Foreman : can view and edit the transaction entries by the Storekeeper and also view reports.

3. Siffs Admin : Siffs admin will have a web application for viewing and managing the users. transactions and create reports.

4. Staff : will have no role in the application
